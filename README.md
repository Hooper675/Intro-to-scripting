# Abbreviation Coder

A simple Python script that checks a user-entered abbreviation and prints its meaning.

## Supported Abbreviations

- LOL: Laugh out loud
- BFN: Bye for now
- FTW: For the win
- IRL: In real life
- JK: Just kidding
- TTYL: Talk to you later
- LMK: Let me know
- NKR: Not work related

## How to Run

```bash
python AbbreviationCoder.py
```

When prompted, enter an abbreviation such as `LOL` or `TTYL`.
